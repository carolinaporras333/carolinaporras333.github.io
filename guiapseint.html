Guía Básica de PSeInt

Esta página explica de forma sencilla las funciones principales de PSeInt, con ejemplos fáciles de entender.

1. Escribir y Leer

Escribir: muestra un mensaje en pantalla.


Leer: permite que el usuario escriba un valor. 

Algoritmo Saludo
  Definir nombre Como Cadena
  Escribir "¿Cómo te llamas?"
  Leer nombre
  Escribir "Hola ", nombre
FinAlgoritmo

2. Variables

Son espacios para guardar datos. Ejemplos:

  • Entero: números sin decimales.
  • Real: números con decimales.
  • Cadena: texto.
Definir edad Como Entero
Definir precio Como Real
Definir nombre Como Cadena

3. Condicionales (Si...Entonces)

Sirven para tomar decisiones.

Algoritmo Nota
  Definir nota Como Real
  Escribir "Ingrese la nota:"
  Leer nota
  Si nota >= 3 Entonces
    Escribir "Aprobado"
  Sino
    Escribir "Reprobado"
  FinSi
FinAlgoritmo

4. Ciclos (Repeticiones)

Permiten repetir instrucciones varias veces.

Algoritmo RepetirNumeros
  Definir i Como Entero
  Para i <- 1 Hasta 5 Hacer
    Escribir "Número: ", i
  FinPara
FinAlgoritmo

5. Funciones y Procedimientos

Sirven para organizar el código.

Algoritmo Ejemplo
  Funcion Sumar(a, b)
    Sumar <- a + b
  FinFuncion

  Escribir "Resultado: ", Sumar(2,3)
FinAlgoritmo

Resumen

En PSeInt lo más importante es:

  • Escribir → mostrar información.
  • Leer → pedir datos.
  • Variables → guardar datos.
  • Si → tomar decisiones.
  • Para, Mientras, Repetir → repetir instrucciones.
